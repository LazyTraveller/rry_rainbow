#### 项目启动

  ```
    # 安装依赖
    yarn

    # 各个包的依赖安装
    yarn storybook

    # 项目启动
    yarn storybook

    #发布
    yarn build-storybook
    
    # 发布到chromatic
    yarn chromatic
  ```
