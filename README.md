#### 项目启动

  ```
    # 安装依赖
    yarn

    # 各个包的依赖安装
    yarn storybook

    # 项目启动
    yarn storybook

    #发布
    yarn build-storybook
    
    # 发布到chromatic
    yarn chromatic

    # 发布到npm 
    npm login
    whoami 查看当前用户
    npm publish

    # 发布到npm的包名称为rry-ui
    可使用npm install rry-ui
    或者使用yarn add rry-ui 
    进行安装使用或测试
  ```
